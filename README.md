### 安装giffgaff app和HookEuicc
10. 安装giffgaff和hookeuicc，打开hookeuicc的esim伪装凭证
![图片说明](img/image10-1.png)

11. 打开lsp框架，启用模组

12. 打开giffgaff 去登录，国内需要翻墙才能登陆giffgaff，自己下载一个翻墙软件放到共享文件夹安装到模拟器就行，我这里装的是clashmeta。由于我的账号注册过esim，页面会不太一样，但是可以看到登录后不支持esim，因为上一步没有重启模拟器，配置没生效。重启下模拟器后，可以看到手机支持esim了。
![图片说明](img/image12-1.png)

### 申请esim
13. 到这一步就可以去申请esim卡了，选择【choose your plan】
![图片说明](img/image13-1.png)

14. 滑到底部，选择自行充值话费，最低10磅，这里要选付款方式
![图片说明](img/image14-1.png)

15. 付款方式只支持visa和万事达，选完付款方式划到底部去付款
![图片说明](img/image15-1.png)

16. 新账号需要填写地址，用生成器：https://www.meiguodizhi.com/ 随便填一下就行
![图片说明](img/image16-1.png)

17. 保存后提示添加信用卡，填好信息后去支付
![图片说明](img/image17-1.png)

18. 选择继续安装esim，由于模拟器实际是不支持esim的，所以无法安装，这时候会弹出一串esim激活码自动复制到剪贴板。保存下来，后面可写入实体白卡。
![图片说明](img/image18-1.png)

19. 返回giffgaff首页，右上角就是我们的手机号了。第一位0可以不用管，所以完整英区手机号为：+44你的号码
![图片说明](img/image19-1.png)

20. 至此，esim手机号就申请好了，话费是刚刚充值的10磅。手机支持esim的朋友看到这里就可以了，像我一样手机不支持esim的朋友，需要继续跟着教程B往下走。

## 教程B：esim写入实体白卡
### 使用安卓设备写入
1. 在安卓设备插入sim白卡
2. 下载easyeuicc，https://easyeuicc.org/zh-hans/，打开软件进行兼容性检查
![图片说明](img/image-b-2-1.png)

3. 如果是giffgaff提供的激活码，我们需要在激活码前加上 LPA: 再进行第三步。见截图。
![图片说明](img/image-b-3-1.png)

4. 下载配置文件后，就激活了
5. 小米手机激活短信服务会发送一条短信，所以话费余额是9.7磅
![图片说明](img/image-b-5-1.png)

## 保号教程

1. 180天内话费余额变动一次即可保号，成本最低的方式是发送一条短信，0.3磅。可以给自己的英国号码发短信。
2. 切记关闭手机卡的移动流量，巨贵！
